# HPTPhotosViewer #

HPTPhotosViewer is an Album or Gallery app just like the Photos app in iOS.

# Features #
* Album List view and Gallery view for an Album.
* Zoom transition from gallery to individual image.
* Pan and Zoom the idvidual image
